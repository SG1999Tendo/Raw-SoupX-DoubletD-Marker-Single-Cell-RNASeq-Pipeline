This Code is for SingleCell RNASeq. Sometimes, the sample can have both raw barcodes and matrixes as well as filtered ones. Normally, it makes sense to just use the filtered ones, but an important internal validation is to use the raw files as well to find rare cell types. This code does both (filtered and raw) in parallel with the raw files using SoupX + DoubletFinder while filtered using only DoubletFinder. This will be updated with time.
